柔術シミュレータ — 遊び方ガイド

Stage 1 プロトタイプ / M1 スコープ = クローズドガードの攻防のみ

1. このゲームについて

本作はブラジリアン柔術(BJJ)の クローズドガード という局面を再現した対戦シミュレータです。 下になっている側(BOTTOM)は両足で相手の腰をロックしながらスイープや関節技を狙い、 上になっている側(TOP)は姿勢を維持しつつ技を切り、ガードをパスしようとします。

このプロトタイプはロジック検証用の簡易版で、見た目はブロックマンですが、 判定・タイミング・条件は実際の競技ルール(IBJJF/ADCC)を意識した設計になっています。

2. 役割を選ぶ

開始画面で L-Stick 左右(または A / D キー)で BOTTOM / TOP を切り替え、 A ボタン(または Space)で決定します。反対側はAIが自動で動きます。

  • BOTTOM(攻め側) — クローズドガードを維持し、スイープや関節技でラウンドを取る。
  • TOP(防御側) — 姿勢を戻し、攻めの手を切り(カット)、最後にガードをパスする。

BOTTOM(攻め)の意図

  • L-Stick 左右 = 腰を横に振る(ハサミスイープの軸作り)
  • L-Stick 上 = 腰を前に突き出す(ヒップバンプ・フラワーの軸作り)
  • R-Stick = グリップのターゲットゾーン選択(↑ COLLAR / ↓ SLEEVE / ← 襟左 / → 袖右 など)
  • F / J = 左右のトリガー。握りの強さ。強く握るほど崩しに寄与し、スタミナを使う
  • R / U = 左右のフック(足)をロック/解除。両足UNLOCKED = ガード崩壊

TOP(防御)の意図

  • L-Stick = 体重を前方/側方へ掛ける(weight_forward / weight_lateral = 姿勢崩しの相殺)
  • F / J = 左右の手で相手の胸・腰・二頭筋・膝を押さえる(基点圧)
  • R / U + R-Stick = カット試行。R-Stickの向きで攻め側のどの手を狙うか決まる
  • V = パスコミット。条件を満たしているときだけ成立する

4. グリップと手の状態(HandFSM)

各手は下記の6状態を遷移します。

状態意味
IDLE胸前の中立位置。次の動作待ち
REACHINGターゲットへ移動中(200–350ms、距離依存)
CONTACT1フレームのみ接触判定
GRIPPED握っている。強度はトリガー値に追従
PARRIED弾かれた瞬間。即座にRETRACTへ
RETRACT中立へ戻り中(150ms)。この間は再REACH不可

CONTACT での分岐ルール:

  1. 相手が同じゾーンを手で防御中 → PARRIED
  2. 直近 400ms 以内に同手×同ゾーンで弾かれた → PARRIED
  3. それ以外 → GRIPPED

つまり「同じ手で同じ場所を連打してもすぐに取れない」。逆手か別ゾーンに切り替えるのが正解です。

5. 姿勢崩し(Posture Break)

攻めの本質は「崩しの条件を作ってから技に入る」こと。崩しは2次元ベクトル (lateral, sagittal) で表されます。

  • lateral(横) = -1(左へ崩す) 〜 +1(右へ崩す)
  • sagittal(前後) = -1(後傾)〜 +1(前傾、顔が膝に近づく)

崩しに寄与する要素:

  • 攻め側の腰入力(L-Stick の方向×押し込み量)
  • GRIPPED 中の手の引き寄せ(ゾーンごとに方向ベクトルが決まっている)
  • 防御側の weight_forward / weight_lateral は逆方向に相殺する
  • 何もしない時間 — 時定数 800ms で原点に戻る

6. 判断窓(Judgment Window)

発動条件を満たすと 判断窓 が開きます。画面が黄色くパルスし、スローモーション(×0.3)になる時間です。

状態時間タイムスケール
CLOSED不定100%
OPENING0.2s100% → 30%
OPEN最大 1.5s30%
CLOSING0.3s30% → 100%
クールダウン0.4s100%(新規OPENING不可)

重要: 入力は常に実時間(60Hz)でポーリングされるので、スローモ中も指の反応は普通どおり効きます。 迷ったら X(全グリップ解放) で窓を見送ることもできます。

7. カウンター窓(Counter Window)

攻め側の判断窓が開いた瞬間、防御側にも カウンター窓 が開きます(画面は水色)。 攻めの技ごとに対応する返し技があり、タイミング入力で攻めを無効化できます。

攻めの技対応カウンター入力(TOP側)
SCISSOR_SWEEP SCISSOR_COUNTER L-Stick をスイープ方向と逆に最大
TRIANGLE TRIANGLE_EARLY_STACK L-Stick 上 + Space (BTN_BASE) ホールド

カウンター成立時は攻めの判断窓が強制的にCLOSINGへ移行。TRIANGLE_EARLY_STACK は「腕引き出し」フラグも両側リセットします。

8. 技一覧と発動条件

技条件
ハサミスイープ 両足 LOCKED + 左右いずれかの SLEEVE を強度 0.6 以上で GRIPPED + ‖break‖ ≥ 0.4
フラワースイープ 両足 LOCKED + WRIST GRIPPED + 前崩し sagittal ≥ 0.5
三角絞め 片足 UNLOCKED + 「腕引き出し」フラグON + COLLAR GRIPPED
オモプラッタ 片手 SLEEVE GRIPPED + 前崩し 0.6+(向きが同じ)+ 腰回転 |hip_yaw| ≥ π/3
ヒップバンプ 前崩し 0.7+ と腰押し 0.5+ を 300ms 連続維持
十字絞め 両 COLLAR を強度 0.7+ で GRIPPED + ‖break‖ ≥ 0.5

「腕引き出し」について

三角絞めの前提となるフラグ。WRIST / SLEEVE を強度 0.6+ で GRIPPED のまま、hip_pull 方向に 1.5 秒継続して引くと ON になります。 一瞬のグリップでは発生しません。TOPが二頭筋へのベース圧、または RECOVERY_HOLD をかけると強制解除されます。

9. スタミナ

各プレイヤーは stamina: 0〜1 を持ちます。1.0 が完全回復、0.0 が行動不能。

消費

  • 手が REACHING または GRIPPED(強度 0.5+) = -0.02/秒
  • 崩しを維持している状態 = -0.05 × ‖break‖ /秒(攻め側)
  • 技確定 = 一括 -0.1
  • TOPの基点圧/体重配分 = 継続使用で緩やかに消費

回復

  • C(Breath) + 全グリップなし + 腰静止 = +0.1/秒
  • 全四肢 IDLE + グリップなし = +0.03/秒

クランプ(上限制約)

  • stamina < 0.2 — トリガー強度の上限が 0.6(十字絞めなど 0.7+ 条件が満たせない)
  • stamina < 0.05 — 新規 REACHING 不可(既に握っている手は維持できる)

10. 勝敗条件

ラウンドは次のいずれかで終了します:

  • BOTTOM勝利 判断窓で 技を確定(スイープ・関節技いずれか)
  • TOP勝利 条件を満たして パス成立
  • 終了 両足 UNLOCKED = ガードが開く(M1スコープ外のスクランブル扱い)

終了後、画面に結果が表示され、A / Space でセッションをリスタートできます。

11. 上達のヒント

  1. 最初は軽い崩し + 1グリップから。いきなり両グリップを狙うと弾かれて短期記憶で取れなくなります。
  2. 崩しは消えるので、グリップと腰入力をセットで維持すること。
  3. 判断窓はスローモなので焦らない。見送っても 0.4s のクールダウン後にまた狙える。
  4. TOP側のカットは R-Stick の向きが重要。相手の握っている手を狙わないと不発。
  5. スタミナが少ないと高強度技(十字絞めなど)が出せなくなるので、不利なら呼吸で立て直す。
