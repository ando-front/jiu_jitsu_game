## 被験者リクルート時の注意

- 青帯3名は **同一道場内の経験者**が望ましい(柔術体系の多様性を減らしてノイズを抑える)
- 白帯3名は **柔術歴 6ヶ月以上**(用語が通じる)
- 全員に事前に「ガードワークに慣れていること」を条件にする
- 録音同意は任意。同意なしでもセッション実施可
- 所要時間(15分 + 待ち)を事前に伝える
