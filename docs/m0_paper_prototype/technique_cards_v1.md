# 技カード v1.0

**前提**: [m0_protocol_v1.md §2.3](./m0_protocol_v1.md)、条件の原文は [state_machines_v1.md §8.2](../design/state_machines_v1.md) と [input_system_v1.md §D.1.1](../design/input_system_v1.md)。

6技を1枚ずつカード化。セッション中は被験者の前に並べて、いつでも参照できるようにする(暗記課題ではない)。

**印刷想定**: A4 1枚に2枚ずつ(合計3枚)。厚紙 or クリアポケット推奨。

**量子化**: 姿勢崩し値 `0–4`、グリップ強度 `0–2`、腰角度 `-2..+2` は [m0_protocol_v1.md §2.1](./m0_protocol_v1.md) に整合。

---

## ① ハサミスイープ(Scissor Sweep)

```
┌──────────────────────────────────┐
│  ハサミスイープ                    │
│                                    │
│  発動条件:                           │
│  ・両足フック = LOCKED               │
│  ・左右どちらかの袖を強く握る         │
│    (グリップ強度 ≥ 2)              │
│  ・相手の姿勢崩し値 ≥ 2              │
│                                    │
│  決め手(判断窓中の入力):             │
│  ・腰を真横に切る(±1以上)           │
│  ・同時に足フック片方を UNLOCKED へ  │
│                                    │
│  意図:                              │
│  袖で軸を固定しつつ、解いた足で相手の  │
│  膝を刈って横へ転がす                 │
└──────────────────────────────────┘
```

## ② フラワースイープ(Flower Sweep)

```
┌──────────────────────────────────┐
│  フラワースイープ                   │
│                                    │
│  発動条件:                           │
│  ・両足フック = LOCKED               │
│  ・左右どちらかの手首を掴む           │
│    (グリップ強度 ≥ 1)              │
│  ・前崩し(姿勢崩しの前方成分) ≥ 2    │
│                                    │
│  決め手(判断窓中の入力):             │
│  ・腰を前方へ押し込む(+1以上)       │
│                                    │
│  意図:                              │
│  前に落ちかけた相手の手首を掴んで       │
│  花が開くように相手を裏返す            │
└──────────────────────────────────┘
```

## ③ 三角絞め(Triangle Choke)

```
┌──────────────────────────────────┐
│  三角絞め                           │
│                                    │
│  発動条件:                           │
│  ・片足フック = UNLOCKED              │
│  ・相手の腕が「引き出された」状態       │
│    (袖または手首を強く握り続けて       │
│      引いた結果 → GMが3ターン継続で宣言) │
│  ・同側の襟を握っている               │
│    (グリップ強度 ≥ 1)              │
│                                    │
│  決め手(判断窓中の入力):             │
│  ・Aボタン(BTN_BASE)を長押し         │
│                                    │
│  意図:                              │
│  引き出した腕と首を自分の両脚で絞める    │
└──────────────────────────────────┘
```

## ④ オモプラッタ(Omoplata)

```
┌──────────────────────────────────┐
│  オモプラッタ                       │
│                                    │
│  発動条件:                           │
│  ・片手で相手の袖を強く握る           │
│    (グリップ強度 ≥ 2)              │
│  ・前崩し ≥ 3(ほぼ顔が膝に)         │
│  ・握っている袖の側に相手が横崩れ       │
│    (左袖を握る→左横崩し)             │
│  ・自分の腰が大きくひねっている         │
│    (腰角度 |±2|)                   │
│                                    │
│  決め手:                             │
│  ・L-Bumper(左足フック解除)         │
│    + 腰の方向を保持                  │
│                                    │
│  意図:                              │
│  握った袖を軸に自分の足を相手の肩に     │
│  巻きつけて関節を極める               │
└──────────────────────────────────┘
```

## ⑤ ヒップバンプ(Hip Bump / Kimura set-up)

```
┌──────────────────────────────────┐
│  ヒップバンプ                       │
│                                    │
│  発動条件:                           │
│  ・前崩し ≥ 3                       │
│  ・腰の押し込み ≥ 2                  │
│  ・上記の「押し」を 2ターン連続で維持   │
│                                    │
│  決め手:                             │
│  ・腰をさらに前へ + 上体を起こす       │
│                                    │
│  意図:                              │
│  沈みかけた相手を下から衝き上げて       │
│  上下を入れ替える                     │
└──────────────────────────────────┘
```

## ⑥ 十字絞め(Cross Collar Choke)

```
┌──────────────────────────────────┐
│  十字絞め                           │
│                                    │
│  発動条件:                           │
│  ・両方の襟を握る(強度 ≥ 2 両方)    │
│  ・相手の姿勢崩し値 ≥ 2              │
│                                    │
│  決め手:                             │
│  ・両トリガーを最大で保持              │
│    (= 両襟を最大強度で絞る)          │
│                                    │
│  意図:                              │
│  襟を十字に交差させて頸動脈を締める    │
│  ※ M0では「条件成立」のみ確認、      │
│    絞まりきる時間は判定省略             │
└──────────────────────────────────┘
```

## 補足: カードには条件のみ、結果は書かない

各カードは「条件 → 決め手」の順で、**実際に技が極まった時の相手の反応**(タップ/転がる/気絶など)は書かない。M0は判断空間の妥当性検証であり、結果表現はゲーム演出側の責任。

Q-A / Q-B / Q-C([m0_protocol_v1.md §6.2](./m0_protocol_v1.md))のインタビューで条件の妥当性だけを検証する。
